# CodeAlpha Stock Portfolio Tracker

## Description
A simple Python application that calculates the total investment value of a stock portfolio.

## Features
- Displays available stocks
- Accepts stock name and quantity
- Calculates investment value
- Shows portfolio summary
- Saves the result in portfolio.txt

## Technologies Used
- Python
- Dictionary
- Loops
- File Handling

## How to Run

```bash
py stock_tracker.py
```
